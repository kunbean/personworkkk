更新日志 ​

项目版本记录。每次功能变更后必须追加条目。

2024-06-16 — v1.2 在线编辑器完善 ​

新增 ​

  • 格式工具栏：一键插入 Markdown 语法（H1-H3、加粗、斜体、删除线、代码、列表、引用、链接、图片、代码块、表格、分割线）
  • 键盘快捷键：Ctrl+B 加粗、Ctrl+I 斜体、Ctrl+K 链接
  • 简历上传与展示功能 (/resume)
  • 结构化技术文档 (/guide/)

修复 ​

  • 发布失败时错误提示不明确 → 增加 HTTP 状态码中文映射
  • getSha() 静默吞掉错误 → 区分 404（正常）和其他错误（抛出）
  • 中文内容 Base64 编解码问题 → 改用 TextEncoder/TextDecoder 标准 API
  • 删除文章时 SHA 为 null 导致失败 → 增加 null 检查
  • 删除文章后 date/tags 未重置 → 补充重置逻辑
  • 滚动同步除零风险 → 增加范围检查
  • Token 失效后无提示 → 启动时自动验证，失效则退回引导页

变更 ​

  • 站点配色从全粉改为紫/珊瑚/靛蓝渐变（高端多巴胺风格）
  • 去掉页面中的颜文字装饰
  • 发布成功提示从底部小字改为顶部绿色横幅，不自动消失

2024-06-16 — v1.1 在线编辑器 ​

新增 ​

  • 在线 Markdown 编辑器 (/editor)
  • GitHub Token 引导页面（3 步设置流程）
  • 文章列表加载与选择
  • 实时 Markdown 预览
  • 一键发布到 GitHub
  • 在线删除文章

变更 ​

  • 首页从纯静态改为动态特性介绍

2024-06-16 — v1.0 项目初始化 ​

  • 首页 (Hero 布局)
  • 文章列表页 (/posts/)
  • 关于页 (/about)
  • 2 篇示例文章 (hello-world, js-closure)
  • GitHub Actions 自动部署
  • GitHub Pages 托管
  • 暗色模式支持
  • 本地全文搜索
