使用 VitePress 搭建个人博客 ​

📅 2024-01-15 | 🏷️ Vue · VitePress · 博客搭建

为什么选择 VitePress ​

我一直在找一个简单的方式来记录我的技术成长。尝试过多种方案后，最终选择了 VitePress：

方案优点缺点
WordPress功能强大太重，需要服务器
Hexo生态丰富构建速度慢
VitePress极快、简洁、现代相对较新

VitePress 的几个核心优势：

  1. 基于 Vite — 构建速度极快，热更新秒级响应
  2. Markdown 优先 — 专注内容创作，不用关心样式
  3. 内置搜索 — 无需额外配置即可全文搜索
  4. 暗色模式 — 自动适配系统主题

搭建步骤回顾 ​

1. 安装 ​

bash
npm install -D vitepress
1

2. 初始化 ​

bash
npx vitepress init
1

按照提示选择配置项即可。

3. 目录结构 ​

docs/
├── .vitepress/
│   └── config.mts   # 核心配置
├── index.md          # 首页
└── posts/            # 文章目录
1
2
3
4
5

4. 启动开发服务器 ​

bash
npx vitepress dev docs
1

打开 http://localhost:5173 即可预览。

总结 ​

VitePress 让我可以专注于内容本身，不用操心样式、部署等问题。对于技术人员来说，这是记录和分享知识的最佳选择之一。
